# plate-tapping-equipment
Development of touch sensitive USB, Plate Tapping Test equipment for Porto Sports Faculty

In 2011, during my wife's Master in the University of Porto, I've developed an Hardware connected to 2 disk plates to test the dexterity of kids. This was connected to a Borland C++ software that registered the speed of the hands switching between the plates.
